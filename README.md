# 有好租自动拉黑工具

一款基于 Electron 开发的自动拉黑商家桌面应用，支持定时任务和批量管理。

## 功能特性

- 自动登录并保存 Cookie
- 批量管理需要拉黑的商户ID列表
- 每分钟自动执行拉黑任务
- 实时显示任务状态和执行日志
- 本地数据持久化存储

## 安装

```bash
# 克隆项目
cd auto-blacklist

# 安装依赖
npm install

# 启动应用
npm start

# 开发模式（带开发者工具）
npm run dev
```

## 使用说明

### 1. 登录

点击右上角"登录"按钮，在弹出的窗口中完成登录。登录成功后窗口会自动关闭，并显示用户昵称。

### 2. 添加商户ID

在"商户ID列表"文本框中输入需要拉黑的商户信息，每行一个，格式为：

```
商户ID,订单编号
```

例如：
```
10209337,122120701128
10209338,122120701129
10209339,122120701130
```

输入完成后点击"保存列表"。

### 3. 启动自动拉黑

点击"启动自动拉黑"按钮，系统会：
- 立即执行一次拉黑任务
- 之后每分钟自动执行一次

点击"停止"按钮可终止任务。

## 项目结构

```

├── main.js           # Electron 主进程
├── preload.js        # 预加载脚本
├── package.json      # 项目配置
├── renderer/
│   ├── index.html    # 主界面
│   ├── style.css     # 样式文件
│   └── renderer.js   # 渲染进程逻辑
└── services/
    └── api.js        # API 服务
```

## 数据存储

- **Cookie**: `~/Library/Application Support/有好租自动拉黑工具/cookies.json`
- **商户列表**: 浏览器 localStorage

## 注意事项

- 请确保网络连接正常
- 登录 Cookie 有效期可能会过期，需要重新登录
- 建议定期备份商户列表
- 项目启动时，不要翻墙，否则商户助手登录不上
